Despawn Engine

Free Game Engine In-Progress

About the Engine

The Despawn Engine is a work in progress open source voxel game engine based around combining most known optimization techniques, advanced rendering features, without causing unnecessary restrictions on users.

Key Features

  • Open Source
  • Rust + Vulkan(o) Backend
  • LOD Chunks (future)
  • Cubic Chunks (future)
  • RGB Block Lighting (future)
  • Block Game Example (future)
